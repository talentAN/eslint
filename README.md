## eslint-cli

#### 简介

根据项目技术栈(js/ts, react/vue, ...) 一键生成合适的
.eslintrc.js, .prettierrc, babel.config.js 配制文件。

并在 package.json 里添加好对应的 devDependencies, link-stage, husky。

#### 使用方式：

```javascript
// 安装脚手架
npm install @talentan/eslint-prettier-cli -g
// 执行
eslint-cli init [folder]
```
